# Proyecto agregar un producto 🌐
## Funciones principales:
* Persistencia de datos con node
* CRUD completo a través de una API
* Se conecta el front con el backend
* Se usa un formulario para agregar productos
* Se puede eliminar el producto
* Se subio el backend a render
* Se conecto la web con Github pages
## Puntos por mejorar:
* Conexión con una BBDD Mongo o SQL
